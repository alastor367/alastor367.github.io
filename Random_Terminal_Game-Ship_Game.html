Clear function:

here 'def' if defines 'cls()' this fuction will clear terminal if 'debugingtime' is difrent that 1.

Rest of code is trying to get system name if windows use 'cls' else use 'clear'.

def cls():
    debugingtime = 0
    if debugingtime != 1:
        system_name = platform.system()
        if system_name == "Windows":
            os.system("cls")
        else:
            try:
                os.system("clear")
            except Exception as e:
                print("Error while cls function")
